# ü§ù Contributing to Car Knowledge Site

Thank you for your interest in contributing! This project helps beginners learn about cars.

## üéØ How to Contribute

### 1. Report Issues
Found a bug or have a suggestion?
- Check if issue already exists
- Create new issue with clear description
- Include screenshots if applicable

### 2. Improve Content
- Keep language simple (6th-8th grade level)
- Use active voice
- Keep sentences under 20 words
- Add helpful analogies
- Cite sources if adding technical info

### 3. Add Features
Before adding new features:
- Open an issue to discuss
- Keep it simple and accessible
- No dependencies/frameworks
- Test on mobile devices
- Check accessibility (WCAG AA)

### 4. Fix Bugs
- Fork the repository
- Create a feature branch
- Make your changes
- Test thoroughly
- Submit pull request

## üìù Content Guidelines

### Writing Style
- **Clear:** No jargon without explanation
- **Friendly:** Encouraging tone, not condescending
- **Practical:** Focus on actionable information
- **Safe:** Include safety warnings where needed

## üì¶ Pull Request Process

1. **Fork & Clone**
   ```bash
   git clone https://github.com/YOUR-USERNAME/car-knowledge-site.git
   cd car-knowledge-site
   ```

2. **Create Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Changes**
   - Edit files
   - Test locally
   - Check accessibility

4. **Commit**
   ```bash
   git add .
   git commit -m "Add: brief description of changes"
   ```

5. **Push & PR**
   ```bash
   git push origin feature/your-feature-name
   ```
   Then create PR on GitHub
